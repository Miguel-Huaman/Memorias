- ## Clonar un repositorio
- git clone <path>
-
- ### Agregar archivos al repositorio
- git status
  Sirve para revisar el estado.
- git add
  Sirve para agregar archivos
- git add -A
  Agrega todos los archivos
- git commit -m "Descripción del commit"
  Crea un nuevo commit
